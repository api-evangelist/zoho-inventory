# Zoho Inventory (zoho-inventory)

Zoho Inventory is a cloud-based inventory management platform with a REST API for managing items, warehouses, sales orders, purchase orders, shipments, stock adjustments, and financial workflows. The API enables integration with e-commerce, accounting, and logistics systems and supports multi-warehouse operations, serial/batch tracking, and barcode management across global datacenter regions.

## APIs

### Zoho Inventory API

REST API for the Zoho Inventory platform, providing programmatic access to items, item groups, warehouses, sales orders, purchase orders, invoices, bills, shipments, stock adjustments, transfer orders, serial numbers, batch tracking, contacts, price lists, taxes, and more.

- **Human URL:** [https://www.zoho.com/inventory/api/v1/introduction/](https://www.zoho.com/inventory/api/v1/introduction/)
- **Base URL:** `https://www.zohoapis.com/inventory/v1`
